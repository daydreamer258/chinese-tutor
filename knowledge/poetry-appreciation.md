## 2. 常见手法（表现手法 + 修辞手法）

### 2.1 修辞手法

| 手法 | 作用 | 例句 |
|------|------|------|
| **比喻** | 形象生动 | "大弦嘈嘈如急雨" |
| **拟人** | 赋予人的情感 | "感时花溅泪，恨别鸟惊心" |
| **夸张** | 突出强调 | "飞流直下三千尺" |
| **对偶** | 整齐对称 | "两个黄鹂鸣翠柳，一行白鹭上青天" |
| **借代** | 突出特征 | "朱门酒肉臭"（"朱门"代富贵人家） |
| **设问** | 引人思考 | "问君能有几多愁" |
| **反问** | 加强语气 | "人生自古谁无死" |
| **用典** | 含蓄深刻 | "至今思项羽，不肯过江东" |
| **双关** | 含蓄双关 | "东边日出西边雨，道是无晴却有晴" |
| **互文** | 交错见义 | "秦时明月汉时关" |

### 2.2 表达技巧

| 手法 | 说明 | 常见答题句式 |
|------|------|-------------|
| **借景抒情** | 通过景物描写抒发情感 | 借……之景，抒发了……之情 |
| **托物言志** | 借某物表达志向/品格 | 借……（物象），表达了……的志向 |
| **动静结合** | 动态与静态画面相衬 | 动静相衬，使画面更加…… |
| **虚实结合** | 现实与想象/回忆交织 | 实写……，虚写……，虚实结合，拓展意境 |
| **衬托/烘托** | 以 ……衬托 …… | 以乐景写哀情 / 以哀景写乐情 |
| **对比** | 鲜明对照 | 将……与……进行对比，突出…… |
| **白描** | 朴素勾勒，不事雕琢 | 用朴素的语言勾勒出…… |
| **象征** | 以具体物象暗示抽象意义 | ……象征了……品质/精神 |
| **以小见大** | 从细微处见宏大主题 | 从……这件小事，折射出…… |
| **卒章显志** | 结尾点明主旨 | 结尾处……直接点明了……的主旨 |

## 4. 万能答题模板

### 4.1 意象题

> 这首诗通过 **____、____** 等意象，营造了 **____** 的氛围，表达了诗人 **____** 的情感。
>
> 如果问"含义"：**____** 意象表层指 ____，深层指 ____，这里借此表达 ____。

### 4.2 手法/技巧题

> 这首诗运用了 **____** 手法，诗中通过 **____**（具体诗句/内容），写出了 **____**（特点），生动地表现了 **____**（情感/主旨）。

### 4.3 情感题

> 诗歌通过写 **____**（内容概括），借助 **____**（意象/手法），抒发了诗人 **____** 之情，同时也表现出 **____**。

### 4.4 意境题

> 诗歌描绘了 **____**（画面），营造出 **____、____** 的意境，表达了 **____** 情感。
>
> ⚠️ 意境题必须答满三要素：**景 + 境 + 情**

### 4.5 炼字/关键词赏析题

> " **____** "（字/词）意为 **____**，运用了 **____** 手法，写出了 **____** 特点，既描绘了 **____** 画面，又表达了 **____** 情感。

### 4.6 句段赏析题

> 该句写了 **____**（内容），运用 **____**（手法），表达了 **____**（情感），在结构上起到了 **____**（承上启下/铺垫/点题/呼应等）作用。

## 5. 答题流程（考场实用）

1. **读标题** → 判断题材类型（送别/边塞/山水田园/咏史怀古/羁旅思乡……）
2. **看作者** → 联系作者风格与时代背景
3. **抓意象** → 圈出诗中物象，联想情感
4. **析手法** → 判断修辞与表达技巧
5. **品语言** → 关注炼字、诗眼
6. **定情感** → 综合判断主流情感 + 暗含情感
7. **套模板** → 用规范化句式组织答案
