# 语言文字运用：成语 + 病句 + 得体 + 连贯

> 最后更新：2026-06-19
> 考查分值：约 18-20 分

---

## 1. 成语题

### 1.1 考查形式

- "下列各句中，加点成语使用恰当的一项是……"
- "依次填入横线处的成语，最恰当的一组是……"
- 与语段结合，判断/修改使用不当的成语

### 1.2 五大常见错误类型

| 错误类型 | 说明 | 示例 |
|---------|------|------|
| **望文生义** | 按字面乱猜，不理解本义 | "万人空巷"指人都出来了，热闹非凡，不是"巷子空了" |
| **感情色彩不当** | 褒贬混用、庄重/口语冲突 | "无所不为"是贬义（什么都干得出来），不能用于好人好事 |
| **搭配对象不当** | 主语/宾语/修饰对象错了 | "美轮美奂"只能形容建筑，不能形容人物或风景 |
| **语体色彩不符** | 书面语用成了口语，或口语用成了书面语 | |
| **适用范围错误** | 时间、程度、场合用错 | "莘莘学子"本身含"众多"，不能"一个莘莘学子" |

### 1.3 答题步骤

1. 看句意，确定需要表达的准确含义
2. 逐一排除：含义不符 → 感情色彩不对 → 搭配不当 → 语体不合
3. 对"似是而非"的，注意本义 vs 引申义

### 1.4 高频易错成语（部分）

| 成语 | 正确含义 | 常见误解 |
|------|---------|---------|
| 差强人意 | 大体上还能让人满意 | 误解为"让人很不满意" |
| 首当其冲 | 最先受到攻击或遭遇灾难 | 误解为"冲在最前面" |
| 不刊之论 | 不能改动的言论，形容精确 | 误解为"不能刊登的言论" |
| 七月流火 | 天气转凉 | 误解为"天气很热" |
| 明日黄花 | 过时的事物 | 误解为"未来的美好事物" |
| 炙手可热 | 权势大，气焰盛 | 误解为"很受欢迎" |
| 目无全牛 | 技艺极其精湛 | 误解为"看不到整体" |
| 文不加点 | 文思敏捷，一气呵成 | 误解为"不加标点" |

## 2. 病句题

### 2.1 六大语病类型（考纲规定）

| 类型 | 出现频率 | 标志提示 |
|------|---------|---------|
| **搭配不当** | ⭐最高频（约29%） | 主谓、动宾、修饰语与中心词不搭配 |
| **成分残缺/赘余** | ⭐高频（约26%） | 缺主语/宾语/必要介词；同义重复 |
| **结构混乱** | 高频（约21%） | 句式杂糅、前后结构不一致 |
| **语序不当** | 中等（12%） | 定语、状语位置错乱、否定词位置错 |
| **不合逻辑** | 较低（8%） | 自相矛盾、因果颠倒 |
| **表意不明** | 最低（5%） | 指代不清、有歧义 |

### 2.2 各类型识别要点

**搭配不当（最高频）**
- 主谓不搭配："成绩取得了明显提高"（成绩不能"取得提高"）
- 动宾不搭配："我们需要养成良好的学习方法"（方法不可"养成"）
- 一面对两面："能否认真复习是考好的关键"（前面"能否"是两面，后面"考好"是一面）

**成分残缺/赘余**
- 缺主语：介词结构开头导致缺主语 — "通过……使……"
- 缺宾语："完善和建立"（应"完善制度并建立制度"）
- 赘余："提前预先""再次重返""新的创新"

**语序不当**
- 定语远离中心词
- 多项状语顺序混乱（时间→地点→方式）
- "不""没"位置错误

**结构混乱**
- 句式杂糅："原因是……造成的"（"原因是……"or"是由……造成的"二选一）
- 前后换结构："本着……为原则"（"本着……的原则"or"以……为原则"）

**表意不明**
- 指代不清："他哥哥和他的同学"（"他"指谁？）
- "以上""以下""左右"模糊

**不合逻辑**
- 自相矛盾："基本上全部同意"
- 因果颠倒："因为他学习好，所以很努力"

### 2.3 做题顺序

1. **先找主干**（主谓宾），看是否残缺、搭配
2. **再看枝叶**（定状补），看语序和搭配
3. **最后扫逻辑**（关联词、否定词、两面词）

## 3. 得体题

### 3.1 四大得体维度

| 维度 | 核心问题 |
|------|---------|
| **语体得体** | 正式场合 vs 非正式场合，书面语 vs 口语 |
| **对象得体** | 对长辈/上级 vs 对同学/朋友 |
| **身份得体** | 以谁的身份说话，用什么口吻 |
| **礼貌得体** | 称呼、敬语、委婉表达 |

### 3.2 高频失误点

- 把口语/网络语用在正式场合
- 对长辈使用命令式语气
- 以个人身份发表单位立场
- 宣传语中过于绝对、含歧视色彩
- 称呼不当（"令尊"称对方父亲，"家父"称自己父亲，不可混淆）

## 4. 连贯题

### 4.1 常见题型

- 将打乱顺序的句子排序
- 选择最恰当的一句填入空缺处
- 补写句子使语段完整连贯

### 4.2 解题思路

1. **通读全段，理清思路**：把握话题和整体结构（总分/并列/因果/转折/时间顺序）
2. **找"线索词"**：
   - 关联词（因此、但是、不仅……而且……、首先……其次……）
   - 指代词（这、那、其、其中、这些）
   - 时间、空间顺序词
   - 关键词复现或同义词替换
3. **定位首尾**：
   - 首句 → 提出话题/总起句
   - 尾句 → 总结/结果/延伸
   - 具体举例（"比如/例如"）通常在中间
4. **先小范围排序，再大结构验证**

## 5. 新高考趋势

### 5.1 情境化命题
- 语言运用题嵌入真实场景：公告、新闻、科普推文、社区活动、课堂讨论
- 不再孤立考知识点，而是考你在语境中运用语言的能力

### 5.2 "一拖多"形式
- 一段材料挂多道小题，可能同时考成语、病句、连贯等多种考点
- 材料来源于真实生活文本

## 6. 典型真题思路速查

| 题型 | 操作步骤 |
|------|---------|
| 成语选择 | 先看句意→逐一排除 |
| 病句判定 | 找主干→看搭配→扫逻辑 |
| 句子排序 | 找线索词→定首尾→小范围排→整体检查 |
| 补写句子 | 看前后逻辑→判断句子功能→组织语言 |
| 得体判断 | 看对象→看场合→看身份→看语体 |
| 语段修改 | 先语义→再语法→最后语体 |
