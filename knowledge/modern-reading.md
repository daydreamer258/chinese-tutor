# 现代文阅读：论述类 + 文学类 + 实用类

> 最后更新：2026-06-19

---

## 1. 通用做题流程（五步法）

1. **读题干** → 圈关键词、限制词、否定词（"不正确""最恰当""哪些方面"）
2. **回原文定位** → 根据题干信息，回到文中锁定对应区域
3. **比对原文** → 逐句对照，检查是否偷换概念、扩大缩小、强加因果
4. **分点作答** → 主观题按要点分条，答案尽量用原文关键词
5. **检查** → 确认答题完整，没有遗漏要点

---

## 2. 论述类文本

**文本特征**：说理性、逻辑性强，考查论点、论据、论证方法、信息筛选

### 2.1 选择题技巧

- **区分题干问法**：
  - "关于原文内容的表述" → 考信息对应
  - "理解和分析" → 考深层理解
  - "分析和推断" → 考逻辑推理
- **抓关键词定位**：特别注意"原因、结果、因此、但是、并非、全部、部分、可能" — 这些决定选项正误
- **常见设误陷阱**：
  - 偷换概念：把 A 说成 B
  - 扩大/缩小范围：把"部分"说成"全部"
  - 因果倒置：原因和结果颠倒
  - 强加关系：原文无关，选项说有因果
  - 无中生有：原文没有的内容
  - 已然/未然混淆：把"将要"说成"已经"

### 2.2 主观题答题

- **围绕论点**：先找文章中心观点，再看各段如何论证
- **组织模式**：观点 → 依据 → 作用
- **提炼逻辑链**：注意分清总分、并列、递进、因果等结构

### 2.3 常见题型

| 题型 | 答题要点 |
|------|---------|
| 理解重要概念含义 | 回到语境，结合上下文解释 |
| 理解重要句子含意 | 表层含义 + 深层含义（往往是观点/论证要点） |
| 筛选并整合信息 | 分点提炼，合并同类项 |
| 分析文章结构 | 总分/并列/递进/因果，说明各段逻辑关系 |
| 分析论点、论据、论证方法 | 指明论点 + 论据类型 + 论证方法 |
| 概括作者观点态度 | 找议论句、评价性语句 |

## 3. 文学类文本（散文/小说/戏剧）

**文本特征**：形象性、情感性，考查人物、情节、环境、主题

### 3.1 选择题技巧

- **先把握文本整体**：人物、事件、情感基调、叙述视角
- **注意情感与形象**：选项常在人物性格、情感变化上设误
- **防止过度拔高**：不要局部细节当成全文中心

### 3.2 主观题答题

- **核心公式**：**内容 + 手法 + 效果**
- **结合语境**：落到具体语句、段落和上下文
- **术语要精**：用术语但不堆术语，必须结合文本

### 3.3 常见题型

| 题型 | 答题角度 |
|------|---------|
| 概括情节/内容要点 | 时间线/事件线，简要概括 |
| 分析人物形象 | 外貌+语言+动作+心理+他人评价 |
| 赏析句子表达效果 | 手法 + 内容 + 情感 + 结构作用 |
| 分析环境描写作用 | 交代背景 / 渲染气氛 / 推动情节 / 衬托人物 / 暗示主题 |
| 理解标题含义 | 表层含义 + 深层含义（往往是文章主题或文眼） |
| 分析主题或情感变化 | 提炼主题词，结合人物/情节梳理情感线 |
| 鉴赏表达技巧 | 指明手法 → 结合内容 → 分析效果 |

## 4. 实用类文本（新闻/传记/报告/科普/应用文）

**文本特征**：真实性、实用性，考查信息筛选和归纳

### 4.1 选择题技巧

- **抓信息点**：数据、时间、对象、范围
- **防止张冠李戴**：把甲的观点说成乙的
- **注意概括是否准确**：是否遗漏或夸大

### 4.2 主观题答题

- **以信息整合为主**：分散信息归类整理
- **强调准确简明**：用语平实，避免文学化发挥
- **必要时直接摘录关键词**

### 4.3 常见题型

| 题型 | 答题要点 |
|------|---------|
| 概括新闻内容/事件要点 | 5W1H（何人、何事、何时、何地、何故、如何） |
| 筛选关键信息 | 分点列举 |
| 概括材料共同点 | 提取各材料的交集 |
| 概括人物事迹/品质 | 事迹归纳 + 品质提炼 |
| 分析图表与文字关系 | 数据验证观点 / 材料补充说明 |
| 评价现实意义/社会价值 | 观点 + 材料支撑 + 现实联系 |

## 5. 通用答题模板

### 5.1 概括题

> 文中写了……三个方面：一是……；二是……；三是……

### 5.2 原因题

> 原因有……点：一、……；二、……；三、……

### 5.3 作用/好处题

> 从内容上看，……；从结构上看，……；从表达效果看，……

### 5.4 赏析题

> 运用了……手法（明确指出），写出了……（具体内容），表现了……（情感/观点），增强了……（效果）。

### 5.5 理解题

> 结合语境，句中"……"是指……，强调了……。

## 6. 选择题排错口诀

> **一看偷换二看偏，三看倒置四看编。五看强加六看混，七看绝对八看限。**

| 口诀 | 含义 |
|------|------|
| 偷换 | 偷换概念 |
| 偏 | 以偏概全 |
| 倒置 | 因果倒置 |
| 编 | 无中生有 |
| 强加 | 强加关系 |
| 混 | 已然未然混淆 |
| 绝对 | 过于绝对的表述 |
| 限 | 范围、程度限定词错误 |
