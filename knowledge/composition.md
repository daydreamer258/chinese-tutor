# 作文指导：审题 + 立意 + 结构 + 高分技巧 + 常见问题诊断

> 最后更新：2026-06-19

---

## 1. 审题立意

### 1.1 审题四步法

1. **通读材料**，把握整体意思（不要只看局部）
2. **圈关键词句**，找"文眼"——材料的评议性语句往往是命题意图所在
3. **判断命题类型**：材料作文 / 任务驱动型 / 命题作文
4. **选定立意角度**：健康、积极、可落地，由小见大

### 1.2 常用审题方法

| 方法 | 操作 |
|------|------|
| 抓关键句法 | 材料中反复出现或评论性的句子 |
| 由果溯因法 | 材料中的现象，追问"为什么" |
| 多向辐射法 | 从不同角色、不同角度提出观点 |
| 求同分析法 | 多则材料找共同主题 |
| 辩证分析法 | 看到对立统一的两面 |

### 1.3 命题作文扩题法

用追问来具体化抽象题目：
- **增补要素**："责任"→"谁的责任/对谁负责/怎样才算尽责"
- **探究寓意**："桥"→不只是物体，更是连接、沟通
- **追三问**：是什么/为什么/怎么样

## 2. 高分结构框架

### 2.1 议论文三段式（最稳妥）

```
[开头] 引材料 + 亮观点（150字左右）
    ↓
[主体] 分论点1——分析——例证
       分论点2——分析——例证
       分论点3——分析——例证
       （每个分论点独立成段，2-3个即可）
    ↓
[结尾] 回扣标题 + 升华主题（150字左右）
```

### 2.2 段落内部结构

**段首亮观点 → 解释分析 → 举例论证 → 小结论**

### 2.3 段落数量建议

- 5-7 段为佳
- 开头和结尾要短而有力
- 主体段落要有料（分析+例证）

## 3. 高分技巧

### 3.1 开头要快准狠

- **第一段直接亮观点**，不要绕太久
- 可用：引用名言 / 概述材料 / 设问引入
- ⚠️ 避免"自古以来""随着时代的发展"等套话

### 3.2 主体要有层次

- 一个段落只讲一个分论点
- 分论点之间要有逻辑关系（递进>并列）
- 例证要**少而精**，宁可一例深挖，不要三例堆砌

### 3.3 语言要有变化

- 长短句交替
- 适当使用修辞（排比→气势，比喻→形象，设问→思考）
- 避免全篇平铺直叙

### 3.4 例证要典型

- 经典人物 + 当下事例结合
- 不要用"小明""小红"这种编造的人名
- 高考阅卷看的是论证质量，不是案例数量

### 3.5 卷面必须干净

- **字迹清楚、段落分明**
- 修改用一条线划掉即可，不要涂黑块
- 这直接影响印象分

### 3.6 结尾要收束有力

- 回扣标题/主题
- 可适当延伸/升华，但不要空喊口号
- 留有余味比啰嗦更好

## 4. 常见问题诊断（学生作文病状处方）

| 问题 | 诊断 | 处方 |
|------|------|------|
| **偏题跑题** | 没抓住材料核心，立意过大/过散 | 训练审题四步法，每次先写一句话概括论点 |
| **立意空泛** | 只喊口号，没有具体分析 | 每个分论点都要有"为什么"和"怎么体现" |
| **结构松散** | 观点很多，没有主线 | 写前列提纲，确保段与段之间有递进/并列关系 |
| **素材堆砌** | 东拼西凑案例，不服务中心论点 | 删掉无关案例，保证每个例子都能回扣论点 |
| **语言平淡** | 句式单一，没有文采 | 练习句式变换，适当用排比/比喻，多读优秀范文 |
| **卷面凌乱** | 字迹潦草，涂改过多 | 限时训练时也要求卷面，养成打腹稿再动笔习惯 |
| **开头冗长** | 引言绕三圈还没亮观点 | 强制开头前50字内出现中心论点 |
| **结尾草率** | 匆匆结尾或无结尾 | 留3分钟专门写结尾，回扣标题和中心论点 |
| **论证空洞** | "因为……所以……"循环，缺实质内容 | 每段至少配一个具体事例或引用 |
| **思辨不足** | 只会正面论证，不会辩证分析 | 练习"反方观点+驳论"的结构 |

## 5. 2025-2026 备考重点主题

### 5.1 四大高频主题

| 主题 | 细分类别 | 思辨角度 |
|------|---------|---------|
| **科技与人文** | AI、互联网、算法、信息伦理 | 效率 vs 公平、便利 vs 隐私、技术 vs 人性 |
| **传统文化与现代** | 国潮、非遗、经典阅读、文化自信 | 传承 vs 创新、本土 vs 全球、保护 vs 开发 |
| **个体与集体** | 青年担当、规则与自由、个人选择 | 个性 vs 共性、自由 vs 责任、个人 vs 时代 |
| **叙事与意义** | 故事、记忆、日常生活中的价值 | 真实 vs 虚构、记忆 vs 遗忘、局部 vs 整体 |

### 5.2 2025 命题新信号

- **去宏大叙事化**：从具体文本、日常意象切入，避免"假大空"
- **强化递进式思辨**：题目设计多层级追问（是什么→为什么→怎么办）
- **文化自觉渗透**：经典文本化用、传统意象现代转换

## 6. 高效训练建议

1. **练审题比练整篇更重要**：每天拿一道作文题，只做审题和立意，写一句话概括
2. **先写提纲再动笔**：养成提纲习惯，5 分钟搭框架
3. **限时训练**：55 分钟完成审题+写作
4. **积累思辨角度**：建立"关系型思辨"（A与B的关系）常用角度库
5. **素材精用**：10 个精用素材 > 50 个堆砌素材
